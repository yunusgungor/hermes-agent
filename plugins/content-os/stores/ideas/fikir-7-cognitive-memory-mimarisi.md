# Fikir: AI Agent'lar İçin Cognitive Memory Hierarchy

**Kaynak:** Kendi deneyim + GBrain geliştirme süreci
**Rota:** ORIGINAL
**Pillar:** Pillar 4 — Cognitive Engineering

Human-like memory layers'ın LLM sistemlerde modellenmesi:
- Working memory (current context window)
- Episodic memory (session history)
- Semantic memory (knowledge graph / vector store)
- Procedural memory (skills, tools, workflows)

Her katmanın:
- storage mechanism
- retrieval latency
- update strategy
- decay/forgetting policy

Gerçek implementasyon: GBrain'den alınan dersler.
