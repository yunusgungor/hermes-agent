# Pattern Kütüphanesi

> Bu dizin, başarılı içeriklerden çıkarılmış tekrar kullanılabilir pattern'leri içerir.
> Hook kurguları, thread yapıları, teknik analiz şablonları.

## Standart Format

```markdown
# [Pattern Adı]

**Tipi:** Hook / Thread Structure / Technique / Format

## Pattern
[Soyutlanmış açıklama]

## Örnek Kullanım
```[içerik snippet]```

## Ne Zaman Kullanılır
[Uygun bağlam]

## Başarı Metrikleri
[Hangi run'larda kullanıldı, sonuçları]
```
